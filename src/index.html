Memeinator

Memeinator is an iOS app that allows you to easily change regular text into meme text, currently available on the App Store for free. It includes complex, layered text transformations and a keyboard extension. Memeinator is monetized with In-App-Purchases using StoreKit 2. Memeinator's source is robust and well-documented, and open-source.

Memeinator uses Swift, SwiftUI, StoreKit, and KeyboardKit.

The Dumptruck

The Dumptruck is hydroponics automation and testing robot built collaboratively with Davis Bachner. Davis worked on most of the hardware, and I wrote the software. The Dumptruck can support three plants, each with different nutrient schedules, useful for A/B testing. The Dumptruck includes a web interface, scheduler, motor driver, and a pH sensor all running on a Raspberry Pi and an Arduino. The Dumptruck has an advanced, modular and extensible project architecture. More info about The Dumptruck's source code is open-source.

The Dumptruck uses HTML/CSS/JS, Express.js, Python, Raspberry Pi, Arduino, Redis, and more.
