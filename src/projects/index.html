Memeinator

Memeinator is an iOS app that allows you to easily change regular text into meme text, currently available as a beta through TestFlight. It includes complex, layered text transformations and a keyboard extension. Memeinator is monetized with In-App-Purchases using StoreKit 2. Memeinator's source is robust and well-documented and will likely be made open-source after its full release.

Memeinator uses Swift, SwiftUI, StoreKit, and KeyboardKit.

Beta (Opens in TestFlight)

Aer

Aer 1 is an advanced Discord bot that is used for moderation and other administration tasks. It can easily be configured and deployed. It currently runs in a number of servers and is available to self-host.

Aer uses Node.js, Yarn, Discord.js, and PM2.

DiscordKit

DiscordKit is a proof of concept library for interacting with the Discord API. While DiscordKit's API coverage is extremely small and it is surely not ready for use in a production project, it is a robust foundation that could be easily extended to include far more of the API.

DiscordKit uses Swift and Vapor.
